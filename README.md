🍴 Adıyaman Pide & Kebap - Digital QR Menu System
This project is a high-speed, sustainable, and mobile-first digital menu solution developed to facilitate the digital transformation of a traditional local business.

🔗 Live Demo
You can access the live version of the project here:

👉 https://muhammedcanharmansah.github.io/adiyaman-pide-menu/

🚀 Project Purpose
The system is designed to minimize operational costs (printing costs, price update expenses) and provide customers with a modern, hygienic, and contactless user experience. This project demonstrates how technology can solve the real-world needs of a local business.

🛠️ Technical Content & Features
Built from scratch using Vanilla JS and Modern CSS without relying on heavy external libraries (like Bootstrap or jQuery) to ensure maximum performance:

Dynamic Data Management: Menu content is managed through a single MENU_DATA object. This allows for easy updates to product names or prices without diving deep into the UI code.

Smart UI Components:

Scroll Spy & Sticky Nav: A navigation bar that automatically tracks the user's scroll position and allows for quick category switching.

Canvas Announcement System: A high-performance animation system built with HTML5 Canvas for scrolling announcements at the top.

Lazy Loading: Images load only when they enter the viewport, ensuring fast initial load times and data savings.

Google Ecosystem Integration: Direct links to the business's Google Maps location and review page to boost customer engagement.

Smart Popup Mechanism: A non-intrusive "Review us on Google" reminder that triggers after a specific user interaction time.

Mobile Optimization (PWA Ready): Optimized with meta tags to look and feel like a native app when "Added to Home Screen" on iOS and Android.

📈 Technical Achievements
Zero-Cost Hosting: Server costs are completely eliminated by utilizing the GitHub Pages infrastructure.

High Performance: A minimalist code structure ensures instant loading even on slow mobile networks (Lighthouse-optimized).

UX/UI Design: A premium dark-mode aesthetic using gold (#D4AF37) and black contrasts to create a high-end restaurant atmosphere.

📱 Implementation
Customers gain instant access to the menu by scanning QR codes placed on tables. The system eliminates the need for downloading any third-party applications.
