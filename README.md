# 🍴 Adıyaman Pide & Kebap - Digital QR Menu System

This project is a **high-speed, sustainable, and mobile-first** digital menu solution developed to facilitate the digital transformation of a traditional local business.

## 🚀 Project Purpose
The system is designed to minimize operational costs (printing costs, price update expenses) and provide customers with a modern, hygienic, and contactless user experience. This project demonstrates how technology can solve the real-world needs of a local business while maintaining zero maintenance costs.

## 🛠️ Technical Features
Built from scratch using **Vanilla JS and Modern CSS** without relying on heavy external libraries to ensure maximum performance and security:

* **Dynamic Data Management:** Menu content is managed through a centralized `MENU_DATA` object. This architecture allows for lightning-fast updates to product names or prices.
* **Smart UI Components:**
    * **Scroll Spy & Sticky Nav:** A custom navigation bar that tracks the user's scroll position for seamless category switching.
    * **Canvas Announcement System:** A high-performance animation system built with **HTML5 Canvas** for smooth, scrolling top-bar announcements.
    * **Lazy Loading:** Images load only as they enter the viewport, ensuring minimal data usage and rapid initial load times.
* **Google Integration:** Deep links to Google Maps and Google Reviews to boost local SEO and customer engagement.
* **Smart Interaction:** A non-intrusive "Google Review" popup mechanism that triggers based on session time to improve the business's online rating.
* **PWA Optimized:** Mobile-specific meta tags and configurations ensure the menu feels like a native application on both iOS and Android.

## 📈 Technical Achievements
* **Zero-Cost Hosting:** Leveraged **GitHub Pages** infrastructure to eliminate server costs permanently.
* **High Performance:** Optimized for **Lighthouse** scores with a minimalist code structure that loads instantly even on 3G networks.
* **Premium UX/UI:** Designed with a sophisticated dark-mode aesthetic, utilizing Gold (`#D4AF37`) accents to enhance the brand's professional image.

## 📱 Implementation
Customers gain instant access to the menu by scanning QR codes placed on tables. The system requires no app installation, working directly within any mobile browser.
